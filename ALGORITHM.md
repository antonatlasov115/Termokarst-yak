# Алгоритм образования термокарстовых линз

## Обзор

Термокарстовые образования формируются при протаивании многолетнемерзлых пород с высоким содержанием льда. Процесс включает несколько взаимосвязанных этапов.

## Этапы алгоритма

### 1. Инициация (Initiation)

**Триггеры:**
- Повышение температуры воздуха
- Нарушение растительного покрова
- Изменение гидрологического режима
- Антропогенное воздействие

**Условия:**
```
if (air_temp > threshold && vegetation_cover < critical_value) {
    initiate_thermokarst()
}
```

### 2. Протаивание (Thaw)

**Формула Стефана (модифицированная):**

```
thaw_depth = sqrt(k * T_eff * t) * f_veg * f_ice

где:
  k - теплопроводность грунта (Вт/(м·К))
  T_eff - эффективная температура (°C)
  t - время (годы)
  f_veg - фактор растительности (0.65-1.0)
  f_ice - фактор льдистости (0.8-1.0)
```

**Параметры по типам грунта:**

| Тип грунта | k (Вт/(м·К)) | Пористость |
|------------|--------------|------------|
| Глина      | 1.2          | 0.45       |
| Песок      | 2.0          | 0.35       |
| Торф       | 0.5          | 0.80       |
| Суглинок   | 1.5          | 0.50       |
| Ил         | 1.8          | 0.55       |

### 3. Просадка (Subsidence)

**Расчет просадки:**

```
subsidence = thaw_depth * (ice_content * 0.9 + compression_coef)

где:
  ice_content - льдистость (0-1)
  0.9 - коэффициент объемного изменения льда
  compression_coef - сжимаемость грунта
```

**Коэффициенты сжимаемости:**
- Песок: 0.05
- Глина: 0.15
- Суглинок: 0.20
- Ил: 0.25
- Торф: 0.40

### 4. Латеральное расширение (Lateral Expansion)

**Формула расширения:**

```
diameter = depth * base_rate * f_water * f_soil * ln(year + 1)

где:
  base_rate = 0.4 (базовый коэффициент)
  f_water = 1.0 + 0.6 * water_availability
  f_soil - фактор типа грунта (0.7-1.3)
  ln(year + 1) - логарифмический рост
```

**Факторы грунта:**
- Песок: 1.3 (быстрое расширение)
- Ил: 1.2
- Торф: 1.1
- Суглинок: 1.0
- Глина: 0.7 (медленное расширение)

### 5. Стабилизация (Stabilization)

**Критерии стабильности:**

```
is_stable = (
    depth < 15.0 м &&
    diameter < 100.0 м &&
    0.05 < aspect_ratio < 0.6 &&
    growth_rate < 1.0 м³/год
)

aspect_ratio = depth / diameter
```

**Оценка стабильности (0-1):**

```
score = 1.0
if depth > 10.0: score -= 0.3
if diameter > 80.0: score -= 0.2
if aspect_ratio > 0.5 or < 0.1: score -= 0.3
if growth_rate > 5.0: score -= 0.2

final_score = max(0.0, min(1.0, score))
```

## Стадии развития

### Initiation (0-5 лет)
- Быстрое протаивание
- Высокая скорость роста (> 10 м³/год)
- Формирование первичной депрессии

### Active Development (5-20 лет)
- Активное боковое расширение
- Скорость роста 1-10 м³/год
- Увеличение диаметра

### Stabilization (20-50 лет)
- Замедление роста (< 1 м³/год)
- Формирование устойчивой геометрии
- Aspect ratio стабилизируется

### Degradation (> 50 лет)
- Возможное обрушение берегов
- Заполнение водой
- Формирование термокарстового озера

## Факторы влияния

### Климатические
- Температура воздуха (основной фактор)
- Продолжительность теплого сезона
- Осадки и влажность

### Геологические
- Тип грунта
- Льдистость
- Глубина залегания мерзлоты

### Биологические
- Растительный покров (защитный эффект)
- Мохово-торфяной слой

### Гидрологические
- Доступность воды
- Дренаж территории
- Уровень грунтовых вод

## Математическая модель

Полная система уравнений:

```
1. Протаивание:
   h(t) = sqrt(k * T * t) * f_v * f_i

2. Просадка:
   s(t) = h(t) * (ε_ice + ε_comp)

3. Диаметр:
   d(t) = s(t) * α * f_w * f_s * ln(t + 1)

4. Объем:
   V(t) = π * (d(t)/2)² * s(t)

5. Скорость роста:
   dV/dt = V(t) - V(t-1)
```
