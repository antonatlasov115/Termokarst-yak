# Термокарстовые образования Якутии

Модульная система симуляции образования термокарстовых линз в условиях многолетней мерзлоты Якутии.

**Версия:** 0.2.0  
**Дата:** 2026-04-08  
**Научная достоверность:** 9.0/10  
**Статус:** ✅ Все тесты проходят (119/119)

## 🎯 Последние улучшения (2026-04-08)

### 🆕 Портирование из PFLOTRAN (3/3)

1. **Модуль фазовых переходов лед-вода**
   - Модель Painter (2011) для расчета насыщенностей
   - Уравнения состояния для льда
   - Характеристические кривые Van Genuchten
   - 5 тестов ✅

2. **Модуль Richards equation**
   - Однофазный поток воды в ненасыщенной зоне
   - Закон Дарси с учетом гравитации
   - Upwind схема и гармоническое усреднение
   - 5 тестов ✅

3. **Интеграционный модуль мерзлых грунтов**
   - Связка фазовых переходов и потоков воды
   - Учет влияния льда на проницаемость
   - Аккумуляция воды и льда
   - 5 тестов ✅

### ✅ Реализовано ранее (8/8)

1. **Полный энергетический баланс поверхности (SEB)**
   - Явный тепловой поток (H)
   - Скрытый тепловой поток (LE)
   - Боуэновское отношение (β = H/LE)

2. **Улучшенная формула Атласова**
   - β: 0.45 → 0.30 (ослабление влияния растительности)
   - w → w^0.7 (усиление роли льдистости)
   - +f_moisture (новый фактор влажности)

3. **Анализ неопределенности**
   - Монте-Карло симуляции (1000+)
   - 90% доверительные интервалы
   - Анализ чувствительности параметров

4. **2D теплоперенос**
   - Метод конечных разностей
   - Латеральные термические процессы
   - Проверка устойчивости

5. **Новые стадии термокарста**
   - Логика на основе глубины
   - Initiation → Active → Stabilization → Degradation

### 📊 Обновленная формула

```
ξ_A = √(2λₜ·DDT / (L·ρw·w^0.7)) · exp(0.30·(1-V)) · (1 + 0.12·ln(ΔT/40)) · (1 + 0.3·water)
```

## Возможности

### Физическое моделирование
- **Протаивание**: расчет глубины по улучшенной формуле Атласова
- **Фазовые переходы**: модель Painter (2011) для лед-вода переходов
- **Потоки воды**: уравнение Ричардса для ненасыщенной зоны
- **Мерзлые грунты**: интегрированная модель с учетом льда

### Энергетика и теплоперенос
- **Полный энергетический баланс**: явный + скрытый + грунтовый тепловые потоки
- **2D теплоперенос**: латеральные термические процессы (метод конечных разностей)
- **Снежный покров**: изоляция, альбедо, сублимация

### Анализ и симуляция
- **Анализ неопределенности**: Монте-Карло симуляции, доверительные интервалы
- **Геологические процессы**: просадка грунта, боковое расширение, консолидация
- **Анализ стабильности**: 4 стадии развития термокарста, оценка рисков
- **Региональные параметры**: предустановки для северной, центральной и южной Якутии
- **Параллельная обработка**: батч-симуляция множества сценариев
- **Работа с данными**: импорт наблюдений, калибровка модели, валидация (6 метрик)

## Установка

```bash
cd thermokarst-yakutia
cargo build --release
```

## Использование

### Одиночная симуляция

```bash
# Симуляция для центральной Якутии на 50 лет
cargo run --release -- simulate -r central -y 50 -o results.json

# Детальный вывод
cargo run --release -- simulate -r north -y 100 -v
```

### Батч-симуляция

```bash
# Параллельная симуляция всех регионов
cargo run --release -- batch -y 50 -o results/ -p
```

### Анализ результатов

```bash
cargo run --release -- analyze -i results.json
```

### Создание конфигурации

```bash
cargo run --release -- config -o my_config.json
```

### Работа с данными наблюдений

```bash
# Создать пример датасета
cargo run --release -- dataset create -o observations.json

# Показать информацию о датасете
cargo run --release -- dataset info -i observations.json

# Калибровать модель по данным
cargo run --release -- dataset calibrate -i observations.json -o params.json
```

## Регионы

- `north` / `северная` - Северная Якутия (высокая льдистость, торф)
- `central` / `центральная` - Центральная Якутия (умеренные условия)
- `south` / `южная` - Южная Якутия (меньшая льдистость)

## Алгоритм

### 1. Инициация
Нарушение теплового баланса (потепление, удаление растительности)

### 2. Протаивание
Расчет глубины по модифицированной формуле Стефана:
- Учет теплопроводности грунта
- Влияние растительного покрова
- Фактор льдистости

### 3. Просадка
Деформация грунта при таянии льда:
- Объем растаявшего льда
- Сжимаемость грунта
- Тип почвы

### 4. Латеральное расширение
Боковой рост термокарста:
- Логарифмический рост
- Влияние доступности воды
- Свойства грунта

### 5. Стабилизация
Оценка устойчивости:
- Геометрические параметры
- Риск обрушения
- Прогноз развития

## Структура крейтов

### `thermokarst-core`
Базовые типы данных:
- `SoilType` - типы грунтов
- `EnvironmentParams` - параметры среды
- `ThermokarstLens` - термокарстовая линза
- `SimulationResult` - результаты симуляции

### `thermokarst-physics`
Физические расчеты:
- `ThawDepthCalculator` - глубина протаивания (формула Атласова)
- `PhaseTransitionCalculator` - фазовые переходы лед-вода (PFLOTRAN) 🆕
- `RichardsCalculator` - потоки воды (уравнение Ричардса, PFLOTRAN) 🆕
- `FrozenGroundCalculator` - интегрированная модель мерзлых грунтов 🆕
- `HeatTransfer2D` - 2D теплоперенос
- `FullEnergyBalance` - полный энергобаланс
- `SubsidenceCalculator` - просадка грунта
- `SnowModel` - модель снежного покрова

### `thermokarst-geology`
Геологические процессы:
- `LateralExpansionCalculator` - боковое расширение
- `StabilityAnalyzer` - анализ стабильности (обновлено!)
- `ConsolidationCalculator` - консолидация грунта

### `thermokarst-simulation`
Движок симуляции:
- `SimulationEngine` - основной движок
- `BatchSimulator` - параллельная обработка
- `UncertaintyAnalyzer` - анализ неопределенности (новое!)
- `ModelCalibrator` - калибровка модели

### `thermokarst-cli`
Интерфейс командной строки с подкомандами:
- `simulate` - одиночная симуляция
- `batch` - батч-обработка
- `analyze` - анализ результатов
- `config` - создание конфигурации
- `dataset` - работа с данными наблюдений
