# История изменений

## Версия 0.1.0 (2026-04-08)

### Первый релиз - Production Ready ✓

#### Основные возможности

**Модульная архитектура**
- 5 независимых крейтов (core, physics, geology, simulation, cli)
- Чистое разделение ответственности
- Возможность использования как библиотеки

**Алгоритм моделирования**
- Расчет глубины протаивания (модифицированная формула Стефана)
- Просадка грунта при таянии льда
- Боковое расширение термокарста
- Анализ стабильности и рисков
- Прогноз времени до стабилизации

**Физические модели**
- `ThawDepthCalculator` - протаивание мерзлоты
- `HeatTransferCalculator` - теплоперенос в грунте
- `SubsidenceCalculator` - просадка поверхности
- `LateralExpansionCalculator` - боковое расширение
- `StabilityAnalyzer` - анализ устойчивости

**Региональные параметры**
- Северная Якутия (высокая льдистость, торф)
- Центральная Якутия (умеренные условия)
- Южная Якутия (меньшая льдистость)

**CLI команды**
- `simulate` - одиночная симуляция с выбором региона
- `batch` - параллельная батч-обработка всех регионов
- `analyze` - детальный анализ результатов
- `config` - создание конфигурационных файлов
- `dataset` - работа с данными наблюдений (create/info/calibrate)

**Работа с данными**
- Импорт/экспорт JSON формата
- Статистический анализ наблюдений
- Калибровка параметров модели
- Валидация по реальным данным
- Фильтрация по географическим регионам

**Документация**
- README.md - основная документация
- ALGORITHM.md - математическая модель (6.2 KB)
- EXAMPLES.md - примеры использования (5.6 KB)
- ARCHITECTURE.md - архитектура проекта (9.5 KB)
- DATA_SOURCES.md - источники открытых данных (7.3 KB)
- DATA_USAGE.md - руководство по работе с данными (8.0 KB)
- PROJECT_SUMMARY.md - краткий обзор (5.4 KB)

#### Технические характеристики

**Статистика**
- Файлов кода: 35
- Строк кода: ~2500+
- Тестов: 14 (все пройдены)
- Размер бинарника: 1.4 MB
- Время компиляции: ~4 сек (release)

**Производительность**
- Одиночная симуляция (50 лет): < 1 мс
- Батч-симуляция (3 региона, 50 лет): < 10 мс (параллельно)
- Анализ результатов: < 1 мс
- Калибровка модели: < 5 мс

**Зависимости**
- serde 1.0 - сериализация
- serde_json 1.0 - JSON формат
- thiserror 1.0 - обработка ошибок
- nalgebra 0.33 - линейная алгебра
- rayon 1.10 - параллелизм
- clap 4.5 - CLI парсинг
- anyhow 1.0 - error handling

#### Тестирование

**Unit тесты (14 тестов)**
- core: типы данных и датасеты (3 теста)
- physics: протаивание, теплоперенос, просадка (5 тестов)
- geology: расширение, стабильность (3 тестов)
- simulation: движок, батч-обработка, калибровка (3 теста)

**Интеграционные тесты**
- Полный цикл симуляции
- Батч-обработка регионов
- Калибровка по данным
- Валидация модели

#### Известные ограничения

- Калибровка использует простые статистические методы
- Не учитывается временная динамика в данных
- Предполагается однородность региона
- Требуется ручная настройка для сложных случаев
