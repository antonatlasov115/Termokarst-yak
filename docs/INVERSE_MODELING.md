## 🆕 Новые возможности

### 1. Обратное моделирование (Inverse Modeling)

Определение времени образования термокарста по текущим наблюдаемым параметрам.

#### Принцип работы

Используя текущие измерения (глубина, диаметр), система восстанавливает:
- **Возраст термокарста** - сколько лет назад он начал формироваться
- **Дату начала** - примерный год образования
- **Параметры среды** - условия, которые привели к текущему состоянию

#### Методы оценки

1. **По глубине протаивания** - обратная формула Атласова
2. **По латеральному расширению** - обратная формула роста диаметра
3. **Комбинированная оценка** - взвешенное среднее (60% глубина + 40% диаметр)

### 2. Интеграция со спутниковыми данными

Модуль для работы с данными дистанционного зондирования:
- Sentinel-2 (10м разрешение)
- Landsat 8/9 (30м разрешение)
- Google Earth Engine
- Planet Labs (3-5м разрешение)

## 🔬 Научная основа

### Обратная формула Атласова

Прямая формула:
```
ξ = √(2λₜ·DDT / (L·ρw·w^0.7)) · exp(0.30·(1-V)) · (1 + 0.12·ln(ΔT/40))
```

Обратная формула (для оценки возраста):
```
t = DDT / DDT_год
где DDT = ξ² · L·ρw·w^0.7 / (2λₜ · факторы²)
```

### Обратная формула латерального расширения

Прямая:
```
D(t) = D₀ + k·ln(1 + t)
```

Обратная:
```
t = exp((D - D₀)/k) - 1
```

## 📊 Источники данных

### Измерения с земли

- **Глубина** - батиметрия, георадар, бурение
- **Диаметр** - GPS измерения, тахеометр
- **NDVI** - полевые спектрометры или спутниковые данные

### Спутниковые данные

#### Sentinel-2 (Рекомендуется)
- **Разрешение:** 10м (RGB, NIR), 20м (SWIR)
- **Частота:** каждые 5 дней
- **Доступ:** https://scihub.copernicus.eu/
- **API:** Sentinel Hub
- **Python:** `sentinelsat`, `eolearn`

```python
# Пример загрузки Sentinel-2
from sentinelsat import SentinelAPI

api = SentinelAPI('user', 'password', 'https://scihub.copernicus.eu/dhus')
products = api.query(
    area='POLYGON((129.2 62.4, 129.4 62.4, 129.4 62.6, 129.2 62.6, 129.2 62.4))',
    date=('20200601', '20200831'),
    platformname='Sentinel-2',
    cloudcoverpercentage=(0, 30)
)
```

#### Google Earth Engine (Лучший выбор)
- **Преимущества:** облачная обработка, исторические данные с 2015
- **Доступ:** https://earthengine.google.com/
- **Python:** `earthengine-api`

```python
# Пример GEE
import ee
ee.Initialize()

# Загрузка Sentinel-2 для области
point = ee.Geometry.Point([129.3, 62.5])
collection = ee.ImageCollection('COPERNICUS/S2') \
    .filterBounds(point) \
    .filterDate('2020-06-01', '2020-08-31') \
    .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 30))

# Расчет NDVI
def add_ndvi(image):
    ndvi = image.normalizedDifference(['B8', 'B4']).rename('NDVI')
    return image.addBands(ndvi)

collection = collection.map(add_ndvi)
```

## 🎯 Практические кейсы

### Кейс 1: Датировка термокарста по спутниковым снимкам

**Задача:** Определить когда образовался термокарст

**Шаги:**
1. Найти самый ранний снимок где виден термокарст (Google Earth Engine)
2. Измерить диаметр на снимке
3. Измерить текущий диаметр (свежий снимок)
4. Запустить обратное моделирование

```bash
# Текущее состояние (2025)
cargo run --release -- inverse \
  --depth 3.5 \
  --diameter 15.0 \
  --latitude 62.5 \
  --longitude 129.3 \
  --year 2025

# Результат: образовался ~1980 год
```

### Кейс 2: Валидация модели

**Задача:** Проверить точность обратного моделирования

**Шаги:**
1. Найти термокарст с известной датой образования
2. Измерить текущие параметры
3. Запустить обратное моделирование
4. Сравнить с реальной датой

### Кейс 3: Массовый анализ региона

**Задача:** Оценить возраст всех термокарстов в регионе

**Шаги:**
1. Детектировать все термокарсты на спутниковых снимках
2. Измерить параметры каждого
3. Запустить обратное моделирование для каждого
4. Построить карту возрастов

## ⚠️ Ограничения

### Текущие ограничения

1. **Упрощенная формула** - не учитывает все факторы
2. **Неопределенность** - ±20% для большинства случаев
3. **Требует калибровки** - для каждого региона свои параметры

### Факторы неопределенности

- Изменение климата со временем
- Антропогенное воздействие
- Локальные особенности грунта
- Точность измерений
