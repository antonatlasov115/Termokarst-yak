## Что было создано

### 1. Модульная архитектура на Rust

Проект организован как Cargo workspace с 5 независимыми крейтами:

```
thermokarst-yakutia/
├── crates/
│   ├── core/          # Базовые типы, датасеты, ошибки
│   ├── physics/       # Физические расчеты (протаивание, теплоперенос, просадка)
│   ├── geology/       # Геологические процессы (расширение, стабильность)
│   ├── simulation/    # Движок симуляции, батч-обработка, калибровка
│   └── cli/           # Интерфейс командной строки (5 команд)
```

### 2. Научно обоснованный алгоритм

**5 этапов моделирования:**

1. **Инициация** - нарушение теплового баланса
2. **Протаивание** - расчет глубины по модифицированной формуле Стефана
3. **Просадка** - деформация грунта при таянии льда
4. **Расширение** - боковой рост термокарста (логарифмический)
5. **Стабилизация** - оценка устойчивости и рисков

**Учитываемые факторы:**
- Теплопроводность грунта (5 типов)
- Льдистость (0-100%)
- Растительный покров (защитный эффект)
- Доступность воды (ускоряет процесс)
- Температурный режим
- Продолжительность теплого сезона

### 3. Региональные параметры для Якутии

**Северная Якутия (70-73°N):**
- Температура: 3°C
- Льдистость: 85%
- Грунт: торф
- Результат (50 лет): 3.29м × 8.41м = 182.5 м³

**Центральная Якутия (60-65°N):**
- Температура: 5°C
- Льдистость: 70%
- Грунт: суглинок
- Результат (50 лет): 6.82м × 14.58м = 1137.8 м³

**Южная Якутия (55-60°N):**
- Температура: 7°C
- Льдистость: 50%
- Грунт: суглинок
- Результат (50 лет): 7.43м × 15.19м = 1346.5 м³

### 5. CLI с 5 командами

```bash
# Симуляция
thermokarst simulate -r central -y 50 -o results.json

# Батч-обработка
thermokarst batch -y 50 -o results/ -p

# Анализ

# Конфигурация

# Работа с данными

thermokarst dataset info -i data.json
thermokarst dataset calibrate -i data.json -o params.json
```

## Примеры использования

### Базовый workflow

```bash
# 1. Создать датасет наблюдений
./target/release/thermokarst dataset create -o observations.json

# 2. Посмотреть статистику
./target/release/thermokarst dataset info -i observations.json

# 3. Калибровать модель
./target/release/thermokarst dataset calibrate -i observations.json -o params.json

# 4. Запустить симуляцию
./target/release/thermokarst simulate -r central -y 50 -o results.json

# 5. Проанализировать результаты
./target/release/thermokarst analyze -i results.json
```

### Батч-обработка

```bash
# Параллельная симуляция всех регионов
./target/release/thermokarst batch -y 50 -o results/ -p

# Результаты:
# - results/Северная_Якутия.json
# - results/Центральная_Якутия.json
# - results/Южная_Якутия.json
```
