# Проект: Термокарстовые образования Якутии

## Краткое описание

Модульная система на Rust для симуляции образования термокарстовых линз в условиях многолетней мерзлоты Якутии.

## Быстрый старт

```bash
# Сборка
cargo build --release

# Простая симуляция
./target/release/thermokarst simulate -r central -y 50

# Батч-симуляция всех регионов
./target/release/thermokarst batch -y 50 -o results/ -p

# Анализ результатов
./target/release/thermokarst analyze -i results/Центральная_Якутия.json
```

## Ключевые возможности

### Физическое моделирование
- Расчет глубины протаивания (формула Стефана)
- Теплоперенос в грунте
- Просадка при оттаивании льда

### Геологические процессы
- Боковое расширение термокарста
- Анализ стабильности
- Оценка рисков обрушения

### Региональные параметры
- Северная Якутия (высокая льдистость, торф)
- Центральная Якутия (умеренные условия)
- Южная Якутия (меньшая льдистость)

### Производительность
- Параллельная обработка (rayon)
- Быстрые расчеты (< 1 мс на симуляцию)
- Эффективная память

## Алгоритм (кратко)

1. **Инициация** - нарушение теплового баланса
2. **Протаивание** - расчет глубины по формуле Стефана
3. **Просадка** - деформация грунта при таянии льда
4. **Расширение** - боковой рост термокарста
5. **Стабилизация** - оценка устойчивости

## Команды CLI

```bash
# Симуляция
thermokarst simulate -r <region> -y <years> [-o output.json] [-v]

# Батч-обработка
thermokarst batch -y <years> [-o dir/] [-p]

# Анализ
thermokarst analyze -i <input.json>

# Конфигурация
thermokarst config -o <output.json>
```

## Использование как библиотеки

```rust
use thermokarst_core::EnvironmentParams;
use thermokarst_simulation::{SimulationConfig, SimulationEngine};

let params = EnvironmentParams::northern_yakutia();
let config = SimulationConfig { years: 50, ..Default::default() };
let engine = SimulationEngine::new(params, config);
let result = engine.run().unwrap();
```
