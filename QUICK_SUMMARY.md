## Примеры использования

### Фазовые переходы
```rust
let calc = PhaseTransitionCalculator::new(...);
let (sats, _, _) = calc.compute(1.0e5, -5.0);
// sats.liquid ≈ 0.4, sats.ice ≈ 0.5, sats.gas ≈ 0.1
```

### Потоки воды
```rust
let calc = RichardsCalculator::new(...);
let flux = calc.flux(&auxvar_up, &material, &auxvar_dn, ...);
```

### Мерзлые грунты
```rust
let calc = FrozenGroundCalculator::new(...);
let state = calc.update_state(1.0e5, -2.0);
let flux = calc.compute_flux(&state_up, &state_dn, ...);
```
