# Tauri Desktop App - Термокарст Якутии

Desktop приложение для симуляции термокарстовых образований с Rust бэкендом и React фронтендом.

## Возможности

- ✅ **Прямая симуляция** - прогноз развития термокарста на 10-100 лет
- ✅ **Обратная симуляция** - определение возраста существующих термокарстовых образований
- ✅ **Интерактивная карта** - выбор местоположения и визуализация на спутниковых снимках
- ✅ **Измерение водоемов** - инструмент для измерения диаметра существующих озер
- ✅ **Батиметрия** - визуализация глубины термокарстовых озер
- ✅ **Графики развития** - динамика глубины и диаметра во времени
- ✅ **Экспорт данных** - сохранение результатов в JSON

## Технологии

**Frontend:**
- React 18 + TypeScript
- Vite
- Leaflet (карты)
- Recharts (графики)
- Lucide React (иконки)

**Backend:**
- Rust + Tauri 2.0
- thermokarst-core, thermokarst-physics, thermokarst-simulation
- Формула Атласова v0.3.0 (w^0.7, β=0.30)

## Установка

```bash
# Установить зависимости
npm install

# Запустить в режиме разработки
npm run tauri dev

# Собрать production версию

```

## Использование

### 1. Выбор режима

- **Прямая симуляция**: прогноз развития нового термокарста
- **Обратная симуляция**: определение возраста существующего

### 2. Выбор местоположения

- Кликните на карту для выбора точки
- Для обратной симуляции: используйте инструмент измерения для обводки водоема

### 3. Настройка параметров

**Прямая симуляция:**
- Регион (север/центр/юг Якутии)
- Температура (+0°C до +5°C)
- Льдистость (0-100%)
- Растительность (0-100%)
- Период симуляции (10-100 лет)

**Обратная симуляция:**
- Текущая глубина (0.5-10 м)
- Текущий диаметр (2-50 м)
- Год наблюдения (2000-2026)

### 4. Запуск и результаты

- Нажмите "Запустить" для выполнения симуляции
- Просмотрите результаты на карте и графиках
- Используйте слайдер для просмотра развития во времени
- Экспортируйте результаты в JSON

## Tauri команды

```rust
// Прямая симуляция
run_forward_simulation(params: SimulationParams) -> Vec<SimulationResult>

// Обратная симуляция
run_inverse_simulation(params: InverseParams) -> InverseResult

// Информация о системе
get_system_info() -> String
```

## Разработка

### Горячая перезагрузка

```bash
npm run tauri dev
```

Frontend обновляется автоматически при изменении файлов в `src/`.
Backend требует перезапуска при изменении Rust кода.

### Сборка

```bash
# Сборка для текущей платформы
npm run tauri build

# Результат в src-tauri/target/release/bundle/
```

## Иконки

Для генерации иконок приложения:

```bash
# Создайте PNG иконку 1024x1024
# Затем запустите:
npx @tauri-apps/cli icon path/to/icon.png
```

Это создаст все необходимые форматы в `src-tauri/icons/`.

## Известные проблемы

- Большой размер bundle (~700KB) - можно оптимизировать через code splitting
- Leaflet карты требуют интернет соединения для тайлов
