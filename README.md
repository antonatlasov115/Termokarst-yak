# Термокарстовые образования Якутии

Модульная система симуляции образования термокарстовых линз в условиях многолетней мерзлоты Якутии.

## Архитектура

Проект организован как Rust workspace с модульной структурой:

```
thermokarst-yakutia/
├── crates/
│   ├── core/          # Базовые типы и трейты
│   ├── physics/       # Физические расчеты (протаивание, теплоперенос, просадка)
│   ├── geology/       # Геологические процессы (расширение, стабильность)
│   ├── simulation/    # Движок симуляции и батч-обработка
│   └── cli/           # Интерфейс командной строки
└── Cargo.toml
```

## Возможности

- **Физическое моделирование**: расчет глубины протаивания по формуле Стефана
- **Геологические процессы**: просадка грунта, боковое расширение
- **Анализ стабильности**: оценка рисков и прогноз развития
- **Региональные параметры**: предустановки для северной, центральной и южной Якутии
- **Параллельная обработка**: батч-симуляция множества сценариев
- **Работа с данными**: импорт наблюдений, калибровка модели, валидация

## Установка

```bash
cd thermokarst-yakutia
cargo build --release
```

## Использование

### Одиночная симуляция

```bash
# Симуляция для центральной Якутии на 50 лет
cargo run --release -- simulate -r central -y 50 -o results.json

# Детальный вывод
cargo run --release -- simulate -r north -y 100 -v
```

### Батч-симуляция

```bash
# Параллельная симуляция всех регионов
cargo run --release -- batch -y 50 -o results/ -p
```

### Анализ результатов

```bash
cargo run --release -- analyze -i results.json
```

### Создание конфигурации

```bash
cargo run --release -- config -o my_config.json
```

### Работа с данными наблюдений

```bash
# Создать пример датасета
cargo run --release -- dataset create -o observations.json

# Показать информацию о датасете
cargo run --release -- dataset info -i observations.json

# Калибровать модель по данным
cargo run --release -- dataset calibrate -i observations.json -o params.json
```

## Регионы

- `north` / `северная` - Северная Якутия (высокая льдистость, торф)
- `central` / `центральная` - Центральная Якутия (умеренные условия)
- `south` / `южная` - Южная Якутия (меньшая льдистость)

## Алгоритм

### 1. Инициация
Нарушение теплового баланса (потепление, удаление растительности)

### 2. Протаивание
Расчет глубины по модифицированной формуле Стефана:
- Учет теплопроводности грунта
- Влияние растительного покрова
- Фактор льдистости

### 3. Просадка
Деформация грунта при таянии льда:
- Объем растаявшего льда
- Сжимаемость грунта
- Тип почвы

### 4. Латеральное расширение
Боковой рост термокарста:
- Логарифмический рост
- Влияние доступности воды
- Свойства грунта

### 5. Стабилизация
Оценка устойчивости:
- Геометрические параметры
- Риск обрушения
- Прогноз развития

## Структура крейтов

### `thermokarst-core`
Базовые типы данных:
- `SoilType` - типы грунтов
- `EnvironmentParams` - параметры среды
- `ThermokarstLens` - термокарстовая линза
- `SimulationResult` - результаты симуляции

### `thermokarst-physics`
Физические расчеты:
- `ThawDepthCalculator` - глубина протаивания
- `HeatTransferCalculator` - теплоперенос
- `SubsidenceCalculator` - просадка грунта

### `thermokarst-geology`
Геологические процессы:
- `LateralExpansionCalculator` - боковое расширение
- `StabilityAnalyzer` - анализ стабильности

### `thermokarst-simulation`
Движок симуляции:
- `SimulationEngine` - основной движок
- `BatchSimulator` - параллельная обработка

### `thermokarst-cli`
Интерфейс командной строки с подкомандами:
- `simulate` - одиночная симуляция
- `batch` - батч-обработка
- `analyze` - анализ результатов
- `config` - создание конфигурации
- `dataset` - работа с данными наблюдений
