## 🔧 Исправленные проблемы

### 1. Overflow ошибка (ИСПРАВЛЕНО ✅)
**Проблема:** `attempt to subtract with overflow` на строке 160
**Решение:** 
- Добавлен `.min(params.observation_year)` для ограничения возраста
- Использован `saturating_sub()` для безопасного вычитания

### 2. Неправильный расчет диаметра (ИСПРАВЛЕНО ✅)
**Проблема:** Упрощенная формула не учитывала физику процесса
**Решение:**
- Интегрирован `LateralExpansionCalculator` из `thermokarst-geology`
- Учитывается тип грунта (песок/глина/торф/ил)
- Учитывается глубина просадки
- Учитывается доступность воды
- Итеративный поиск для обратной симуляции
