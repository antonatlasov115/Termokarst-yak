## 💡 Детальное описание топ-3 модулей

### 1. Граничные условия (4-6 часов) 🔴

**Что портировать:**
- Нейман BC (заданный поток)
- Смешанные BC
- Источники/стоки
- Временные зависимости

**Файлы для изучения:**
- `condition.F90` (5,734 строки)
- `condition_control.F90`

**Ключевые функции:**
- `FlowConditionCreate`
- `FlowConditionRead`
- `FlowConditionUpdate`
- `FlowSubConditionCreate`

**Применение в термокарсте:**
- Инфильтрация осадков (Нейман BC)
- Таяние снега (источник воды)
- Испарение (сток)
- Сезонные изменения (временные BC)

---

### 2. Newton-Raphson solver (8-12 часов) 🔴

**Что портировать:**
- Newton-Raphson итерации
- Линейные решатели (GMRES, BiCGStab)
- Критерии сходимости
- Line search для устойчивости

**Файлы для изучения:**
- `solver.F90` (1,979 строк)
- `convergence.F90`

**Ключевые функции:**
- `SolverCreateSNES`
- `SolverSetSNESOptions`
- Интеграция с PETSc (можно заменить на Rust библиотеки)

**Альтернатива:**
```rust

    x0: na::DVector<f64>,
    tol: f64,
    max_iter: usize,
) -> na::DVector<f64> {
    // Простая реализация Newton-Raphson
}
```

---

### 3. Адаптивный timestepping (6-8 часов) 🔴

**Что портировать:**
- Адаптивный выбор шага
- Контроль ошибок
- Сокращение при неудаче
- Увеличение при успехе
- Rescue mode

**Файлы для изучения:**
- `timestepper_base.F90` (1,297 строк)
- `timestepper_SNES.F90` (1,118 строк)

**Ключевые параметры:**
```fortran
dt_min, dt_max           ! Границы шага
dt_init                  ! Начальный шаг
time_step_reduction_factor    ! Коэффициент уменьшения
time_step_max_growth_factor   ! Максимальный рост
max_time_step_cuts       ! Максимум сокращений
```

**Применение:**
- Автоматический выбор шага при фазовых переходах
- Устойчивость при резких изменениях температуры
- Эффективность (большие шаги когда возможно)
