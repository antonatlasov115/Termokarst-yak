# 🏔️ Термокарст Якутии - Tauri Desktop App

## Быстрый старт

```bash
cd apps/tauri-app

# Установка зависимостей (первый раз)
npm install

# Запуск в режиме разработки
npm run tauri dev

# Сборка production версии
npm run tauri build
```

## Что реализовано

### ✅ Полный функционал
- **Прямая симуляция** - прогноз развития термокарста на 10-100 лет
- **Обратная симуляция** - определение возраста существующих образований
- **Интерактивная карта** - Leaflet с спутниковыми снимками
- **Измерение водоемов** - инструмент для обводки существующих озер
- **Батиметрия** - визуализация глубины в 5 уровнях
- **Графики** - динамика глубины и диаметра (Recharts)
- **Экспорт** - сохранение результатов в JSON

### 📊 Параметры симуляции

**Прямая:**
- Регион (север/центр/юг Якутии)
- Температура (0-5°C)
- Льдистость (0-100%)
- Растительность (0-100%)
- Период (10-100 лет)

**Обратная:**
- Текущая глубина (0.5-10 м)
- Текущий диаметр (2-50 м, можно измерить на карте)
- Год наблюдения (2000-2026)

### 🎨 UI/UX
- Двухэтапный workflow: выбор местоположения → симуляция
- Переключение между режимами
- Градиентный дизайн
- Адаптивная верстка
- Информационные панели

## Технологии

**Frontend:**
- React 18 + TypeScript (1,281 строк)
- Vite
- Leaflet (карты)
- Recharts (графики)
- Lucide React (иконки)

**Backend:**
- Rust + Tauri 2.0 (217 строк)
- thermokarst-core/physics/geology/simulation
- Формула Атласова v0.3.0 (w^0.7, β=0.30)

## Tauri команды

```rust
// Прямая симуляция
run_forward_simulation(params: SimulationParams) -> Vec<SimulationResult>

// Обратная симуляция  
run_inverse_simulation(params: InverseParams) -> InverseResult

// Информация о системе
get_system_info() -> String
```
