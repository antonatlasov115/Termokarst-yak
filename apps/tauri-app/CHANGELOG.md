# Changelog - Tauri App

## [0.3.0] - 2026-04-09

### Добавлено
- ✅ Полностью функциональное desktop приложение на Tauri 2.0
- ✅ Прямая симуляция термокарста (прогноз на 10-100 лет)
- ✅ Обратная симуляция (определение возраста существующих образований)
- ✅ Интерактивная карта с выбором местоположения
- ✅ Инструмент измерения диаметра водоемов
- ✅ Визуализация батиметрии (глубины)
- ✅ Графики развития (глубина, диаметр)
- ✅ Экспорт результатов в JSON
- ✅ Поддержка спутниковых и обычных карт
- ✅ Региональные настройки (север/центр/юг Якутии)

### Технические детали
- React 18 + TypeScript frontend
- Rust backend с интеграцией thermokarst-core/physics/simulation
- Leaflet для карт
- Recharts для графиков
- Формула Атласова v0.3.0 (w^0.7, β=0.30)

### Исправлено
- Настроена сборка Tauri с правильными зависимостями
- Добавлены placeholder иконки
- Исправлены предупреждения компилятора
- Настроен workspace exclude для Tauri

### Известные ограничения
- Требуется интернет для загрузки карт
- Большой размер bundle (~700KB frontend)
- Иконки приложения - placeholder (требуется создать финальные)

- Запуск приложения: ~2-3 сек
